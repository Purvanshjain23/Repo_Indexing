      *****************  RPG PROGRAM HEADING  ************************
      *
      * SYSTEM:      HOG PRODUCTION
      * PROGRAM:     HPOUTQ
      * TITLE:       RETRIEVE USER'S DEFAULT OUTPUT QUEUE
      * PROGRAMMER:  LEANNE FEDOR
      * CREATED:     05/14/97
      *
      * FUNCTION:  THIS PROGRAM RETRIEVES THE DEFAULT OUTPUT QUEUE FOR
      *            A USER FROM THE USERS FILE AND UPDATES AN LDA VALUE.
      *            THIS PROGRAM IS CALLED FROM CLP HPINIT WHEN THE
      *            USER ACCESSES THE HPS MAIN MENU.
      *
      *            WE ORIGNALLY DEFAULTED THE USER'S OUTPUT QUEUE FROM
      *            THEIR USER PROFILE.  BUT THEY WHINED BECAUSE WHEN THEY
      *            PRINTED 'MAIL' AND FORGOT TO CHANGE THE OUTPUT QUEUE
      *            TO THE LASER PRINTER, THE PRINTING MESSED UP THE
      *            ALIGNMENT ON THE LINE PRINTER FOR THE NEXT PERSON.
      *
      *            SO, THEIR USER PROFILE NOW POINTS TO THE LASER PRINTER
      *            SO THEIR 'MAIL' WILL PRINT OK.  ALL HPS PRINTING MUST
      *            DEFAULT TO THE PRINTER SET UP IN THE 'USERS MASTER FILE'
      *            IN HPS.
      *
      ****************************************************************
      * MODIFICATIONS:
      ****************************************************************
      * DATE      PROGRAMMER
      * 99/99/99
      *
      /EJECT
      ****************************************************************
      * FILE SPECIFICATIONS
      ****************************************************************
      *
     FHSP001    IF   E           K DISK
      *    AUTHORIZED USERS  (KEY: AUUSCD)
      *
      /EJECT
      ****************************************************************
      * TABLE AND ARRAY SPECIFICATIONS
      ****************************************************************
      *
      *
      ****************************************************************
      * INPUT SPECIFICATIONS
      ****************************************************************
      *
      *---------------------------------------------------------------
      * STANDARD PROGRAM STATUS DATA STRUCTURE
      *---------------------------------------------------------------
      *    EXTERNALLY DEFINED AS UTPGFR (RECORD FORMAT: PGMDSR)
     D PGMDS         ESDS                  EXTNAME(UTPGFR)
      *
      *---------------------------------------------------------------
      * STANDARD DATABASE FILE INFORMATION DATA STRUCTURE
      *---------------------------------------------------------------
      *    EXTERNALLY DEFINED AS UTDBFR (RECORD FORMAT: FDBCKD)
     D DBFEED        E DS                  EXTNAME(UTDBFR)
      *
      *---------------------------------------------------------------
      *  LDA
      *---------------------------------------------------------------
      *
     D                UDS
     D  LDOUTQ               401    410
      *
     D                 DS
     D  DUMMYD                 1      1
      *  THE PRECEDING DUMMY DATA STRUCTURE IS ONLY TO KEEP THE I-SPEC
      *  INFO FROM RUNNING OVER INTO THE 'CALC SPECS' ON THE COMPILE
      *  LISTING.  IT SERVES NO OTHER PURPOSE AND CAN BE DELETED
      *  AT ANY TIME!!!
      /EJECT
      ****************************************************************
      * MAIN LINE
      ****************************************************************
      *
      *
     C     XXUSCD        CHAIN     HSP001                             92
     C     *IN92         IFEQ      *OFF                                         IF GOOD USER
     C                   MOVEL     AUOUTQ        LDOUTQ
     C                   ELSE
     C                   MOVE      *BLANK        LDOUTQ
     C                   ENDIF                                                  IF GOOD USER
      *
     C                   SETON                                        LR
      /EJECT
      *---------------------------------------------------------------
      * *INZSR - INITIALIZATION SUBROUTINE
      *---------------------------------------------------------------
      *
     C     *INZSR        BEGSR
      *
      * PARM LISTS
      *
     C     *ENTRY        PLIST
     C                   PARM                    XXUSCD
      *
     C     *LIKE         DEFINE    AUUSCD        XXUSCD
      *
     C                   ENDSR
      /EJECT
