      * This program takes in a Synon date (DTE) outpus a *USA       *
      * date with seperator of '/'.  For example:                    *
      *                                                              *
      * 1191220 is converted to 12/20/2019                           *
      *                                                              *
      ****************************************************************

      *---------------------------------------------------------------
      * PROCEDURE PROTOTYPE
      *---------------------------------------------------------------
     D cvtdate         PR                  EXTPGM('PWJ8UPR')
     D   piDatein                     7  0                                      Synon date format
     D   poDateout                   10A                                        Date in mm/dd/yyyy

     D cvtdate         PI
     D   piDateIn                     7  0                                      Synon date format
     D   poDateOut                   10A                                        Date in mm/dd/yyyy

      *---------------------------------------------------------------
      * CALCULATIONS
      *---------------------------------------------------------------

      /free
        poDateOut = %char(%date(piDateIn:*cymd):*usa/);

        eval *inlr = *on;
      /end-free
