      * FUNCTION: This is a Work With that displays the records in the Farm Feed Ticket Overrides
      *           file.  There are no Restrictors, Positioners, Selectors.
      *
      *           Why did we create this file/function?
      *           .......................................
      *           Debbie Deen/John Davison needed all the Farm 30 uploaded feed to go to
      *           Farm 278 and, ultimately, to be allocated to all the Farm 278 groups.
      *           Debbie created a shared Bin on Farm 278 and tied it to the active Rooms.
      *           I created this new Override file/function that will allow Debbie to set
      *           up any "override" that she wants. The Feed Ticket Upload program will access
      *           the new "override" file and make the "overrides" as it writes to the "editing"
      *           files. (Note: The Staging and Historical Staging files in the Feed Ticket Upload
      *           function will hold the uploaded values....not the override values.)
      *           To determine what overrides were in effect at the time of a particular Feed
      *           Ticket Upload, we are also creating a History Farm Feed Ticket Override file
      *           that lets us keep/see each Create/Revise/Delete of an override record.
