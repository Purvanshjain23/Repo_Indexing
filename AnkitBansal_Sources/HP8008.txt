      * PROGRAM:     HP8008
      * TITLE:       Format CCYYMMDD Date into SYNON Format
      * PROGRAMMER:  LeAnne Fedor
      * CREATED:     03/26/02
      *
      * FUNCTION:  Given an incoming date in CCYYMMDD format, this program formats it
      *            into the SYNON format of CYYMMDD.
      *
      ****************************************************************
      * MODIFICATIONS:
      ****************************************************************
      * DATE      PROGRAMMER
      *
      /EJECT
      ****************************************************************
      * FILE SPECIFICATIONS
      ****************************************************************
      *
      *
      ****************************************************************
      * DEFINITION SPECIFICATIONS
      ****************************************************************
      *
      *
      ****************************************************************
      * CONSTANTS
      ****************************************************************
      *
     D yes             c                   'Y'
     D no              c                   'N'
      *
      *
      ****************************************************************
      * ARRAYS AND TABLES
      ****************************************************************
      *
      ****************************************************************
      * STANDALONE FIELDS
      ****************************************************************
      *
      * Work fields for date manipulation
      *
     D wkyymmdd        s              6  0
      *
      *
      * Parms
      *
     D xxccyymmdd      s              8  0
     D xxsynondt       s              7  0
      *
      *
      ****************************************************************
      * DATA STRUCTURES
      ****************************************************************
      *
      *---------------------------------------------------------------
      * Standard program status data structure
      *---------------------------------------------------------------
      *    externally defined as UTPGFR (record format: PGMDSR)
     D pgmds         esds                  extname(utpgfr)
      *
      *
      * The following is a dummy C-spec to keep the D-specs from
      * running over into the C-specs on the compile listing. It
      * serves no other purpose and can be deleted at any time.
      *
     C                   if        sdpgm = *blank
     C                   endif
      /EJECT
      ****************************************************************
      * Mainline
      ****************************************************************
      *
      * Get the incoming date into the SYNON format.
      *
     C                   z-add     xxccyymmdd    wkyymmdd
     C                   if        xxccyymmdd >= 20000000
     C     wkyymmdd      add       1000000       xxsynondt
     C                   else
     C                   z-add     wkyymmdd      xxsynondt
     C                   endif
      *
     C                   seton                                        lr
      /EJECT
      *---------------------------------------------------------------
      * Initialization subroutine
      *---------------------------------------------------------------
      *
     C     *inzsr        begsr
      *
      * Parm lists
      *
     C     *entry        plist
     C                   parm                    xxccyymmdd
     C                   parm                    xxsynondt
      *
     C                   endsr
      /EJECT
