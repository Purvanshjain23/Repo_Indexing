/*  PROGRAM NAME ----> MAINTAIN CHECK                              */
/*                     CREATE ADJUSTMENT FILE IN QTEMP             */
/*  PROGRAMMER ------> LEANNE FEDOR                                */
/*  DATE ------------> 06/20/94                                    */
/*                                                                 */
/*                                                                 */
/*  PROGRAM PURPOSE -> THIS CL CLEARS THE TEMPORARY ADJUSTMENT     */
/*                     FILE IN QTEMP.                              */
/*                                                                 */
/*                     THIS TEMPORARY ADJUSTMENT FILE IS ONLY      */
/*                     REQUIRED WHEN THE USER IS CREATING A        */
/*                     NEW CHECK BECAUSE THE USER CAN CREATE       */
/*                     ADJUSTMENTS FOR THE CHECK PRIOR TO THE      */
/*                     CHECK BEING WRITTEN.                        */
/*                                                                 */
/*                     SO, ADJUSTMENTS ARE WRITTEN TO QTEMP.       */
/*                     THEN, WHEN THE CHECK IS ACTUALLY CREATED    */
/*                     ADJUSTMENTS ARE UPDATED WITH THE CUSTOMER,  */
/*                     CHECK NUMBER AND THE CHECK DATE             */
/*                     AND THEN COPIED VIA A CLP TO THE ACTUAL     */
/*                     ADJUSTMENT FILE.                            */
/*                                                                 */
/***************************************************************** */
/*  MODIFIED:                                                      */
/*  99/99/99  PROGRAMMER                                           */
/*                                                                 */
/***************************************************************** */

             PGM

             CLRPFM     FILE(QTEMP/HSP065)
             MONMSG     MSGID(CPF0000)


             ENDPGM
