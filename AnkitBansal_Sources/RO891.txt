      * TITLE:       Initialize Print Options
      * PROGRAMMER:  LeAnne Ramsey
      * CREATED:     01/04/08
      *
      * FUNCTION:  This program initializes the LDA with:
      *             1) user's default outq/printer (if the printer in the LDA
      *                is not a valid printer)
      *             2) hold/save/copies values
      *
      ****************************************************************
      * MODIFICATIONS:
      ****************************************************************
      * DATE      PROGRAMMER
      *
      * 12/19/08  LeAnne Ramsey
      *           Changed logic to use DRMPNA-Default Printer instead of
      *           DRAHVN-Printer Device. (Both fields held the same value
      *           in every record in the file; so, this change won't make
      *           any difference.)
      /EJECT
      ****************************************************************
      * FILE SPECIFICATIONS
      ****************************************************************
      *
     Fcadrrel1  if   e           k disk
      *  User profile control
      *
      ****************************************************************
      * DEFINITION SPECIFICATIONS
      ****************************************************************
      *
      *
      ****************************************************************
      * CONSTANTS
      ****************************************************************
      *
     D yes             c                   'Y'
     D no              c                   'N'
      *
      *
      ****************************************************************
      * ARRAYS AND TABLES
      ****************************************************************
      *
      ****************************************************************
      * STANDALONE FIELDS
      ****************************************************************
      *
      * Parms
      *
     D xxerror         s              1
      *
      ****************************************************************
      * DATA STRUCTURES
      ****************************************************************
      *
      *---------------------------------------------------------------
      * Standard program status data structure
      *---------------------------------------------------------------
      *    externally defined as UTPGFR (record format: PGMDSR)
     D pgmds         esds                  extname(utpgfr)
      *
      *---------------------------------------------------------------
      *  LDA
      *---------------------------------------------------------------
      *
     Dlda             uds                  dtaara(*lda)
     D  ldoutq               401    410
     D  ldhold               411    414
     D  ldsave               415    418
     D  ldcopy               419    419  0
      *
      *
      * The following is a dummy C-spec to keep the D-specs from
      * running over into the C-specs on the compile listing. It
      * serves no other purpose and can be deleted at any time.
      *
     C                   if        sdpgm = *blank
     C                   endif
      /EJECT
      ****************************************************************
      * Mainline
      ****************************************************************
      *
     C                   if        xxerror = yes
     C     sdusr         chain     cadrrel1                           92
     C                   if        *in92 = *off
     C                   move      drmpna        ldoutq
     C                   endif
     C                   endif
      *
     C                   if        ldhold <> '*YES' and ldhold <> '*NO '
     C                   move      '*NO '        ldhold
     C                   endif
      *
     C                   if        ldsave <> '*YES' and ldsave <> '*NO '
     C                   move      '*NO '        ldsave
     C                   endif
      *
     C                   z-add     1             ldcopy
      *
     C                   seton                                        lr
      /EJECT

      *
      * Key lists

      *
     C                   endsr
