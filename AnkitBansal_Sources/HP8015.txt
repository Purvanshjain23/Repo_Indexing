     C                   z-add     xxccyymmdd    wkyymmdd

     C                   z-add     wkyymmdd      xxcyymmddi
     C                   endif

      * Retrieve the year/week for this date
      *
     C                   call      'POF3XFR'

     C                   parm      350           xxcocd
     C                   parm                    xxcyymmddi
     C                   parm      0             xxjulian
     C                   parm      0             xxacyr
     C                   parm      0             xxacpe
     C                   parm      0             xxacwk
     C                   parm      *blank        xxday
