      *
     C                     Z-ADD#IB3VA    UB3VA            PERIOD AMT.
     C                     Z-ADD#IB6VA    UB6VA            AMOUNT YTD
     C                     Z-ADD#INVNB    UNVNB            UNITS BUDG CURR
     C                     Z-ADD#INWNB    UNWNB            UNITS BUDG YTD
     C                     Z-ADD#INXNB    UNXNB            UNITS BUDG YTDP
      *
     C                     EXSR UACC3                      ACCUMULATE
      *
     C           UACC3     BEGSR
      *===================================================================
      * UACC3 - ACCUMULATE VALUES FOR REPORT FORMATS - BUDGET
      *===================================================================
     C           *LIKE     DEFN #IB3VA    UB3VA
     C           *LIKE     DEFN #IB6VA    UB6VA
     C           *LIKE     DEFN #INVNB    UNVNB
     C           *LIKE     DEFN #INWNB    UNWNB
     C           *LIKE     DEFN #INXNB    UNXNB
     C                     ADD  UB3VA     UB1
     C                     ADD  UB6VA     UB2
     C                     ADD  UNVNB     UV
     C                     ADD  UNWNB     UW
     C                     ADD  UNXNB     UX
      *===================================================================
     C                     ENDSR
