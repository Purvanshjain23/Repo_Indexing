     ‚* FUNCTION:    This program gets email id from ESND distribution list
     ‚****************************************************************
     ‚* MODIFICATIONS:
     ‚****************************************************************
     ‚****************************************************************
     FESNDDSTP  IF   E           K DISK
      *
     D  wkEmail        S             50
     D  wkUser         S             30    Inz
     D  wkGrp          S             15    Inz('*SYSTEM')
      *
     D                SDS
     D  PGMSGQ           *PROC
     D  WRKSTN               244    253
     D  USERNM               254    263
      *
     C     *Entry        PList
     C                   Parm                    wkEmail
      *
     C     KeyDist       KList
     C                   KFld                    wkGrp
     C                   KFld                    WKUSER
      *
     C                   Eval      wkEmail = *Blanks
     C                   Eval      wkUser = %Lower(USERNM)
     C     KeyDist       Chain     ESNDDSTP
     C                   If        %Found(ESNDDSTP)
     C                   Eval      wkEmail = ED#ADR
     C                   Else
     C                   Eval      wkUser = %Upper(USERNM)
     C     KeyDist       Chain     ESNDDSTP
     C                   If        %Found(ESNDDSTP)
     C                   Eval      wkEmail = ED#ADR
     C                   EndIf
     C                   EndIf
      *
     C                   Eval      *InLr = *On
