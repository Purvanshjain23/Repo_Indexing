      * TITLE:       CLOSED GROUP ANALYSIS REPORT
      * PROGRAMMER:  LEANNE FEDOR
      * CREATED:     07/08/97
      *
      *  FUNCTION:   THIS PROGRAM PRINTS THE DATA THE FOLLOWING
      *              WORKFILES:
      *
      *                HSP332 - GROUP WORKFILE (1 RECORD PER GROUP)
      *                HSP333 - GROUP WORKFILE (MULTIPLE RECORDS PER GROUP)
      *                HSP334 - PHASE WORKFILE (1 RECORD PER PHASE)
      *                HSP335 - PHASE WORKFILE (MULTIPLE RECORDS PER PHASE)
      *
      *               THE WORKFILES ARE BUILT IN RPG PROGRAMS:
      *
      *                HP346 - GROUP WORKFILES
      *                HP347 - PHASE WORKFILES
      *
      *              THIS PROGRAM IS SUBMITTED BY HP447CL WHICH IS
      *              CALLED FROM HP447 THROUGH QCMDEXC.
      *
      *              THIS PROGRAM PRINTS ONE COLUMN OF DATA PER GROUP.
      *              THE MAXIMUM COLUMNS ON A PAGE IS 12.
      *              ALL 12 COLUMNS ARE BUILT BEFORE A PAGE IS
      *              PRINTED. AND, IF THERE ARE MORE THAN 12 GROUPS,
      *              ADDITIONAL PAGES WILL BE PRINTED.
      *
