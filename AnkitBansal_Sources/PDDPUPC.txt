/*******************************************************************/
/*      THIS PROGRAM CLEARS ANY FILE IN QTEMP                      */
/*******************************************************************/
PDDPUPC:  PGM PARM(&FILE)

          DCL &FILE  TYPE(*CHAR) LEN(10)

          CLOF       OPNID(&FILE)
          CLRPFM QTEMP/&FILE

EXIT:  ENDPGM
