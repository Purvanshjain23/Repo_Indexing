      * Function: This program allows the users to DWL costs from their Spreadsheets
      *           to directly update the cost file POAXCPP.
      *           The "costs" will come from the user's file/
      *           spreadsheet. They get the costs into file POAXCPDWL which this
      *           program reads updating the SYNON file 'Item Standard Cost'.
      *
      *           There is a submission screen for this program which is on the
      *           menu.

      *************************************************************************************
      * Mainline
      *************************************************************************************
      *
      * Read each record in the Uploaded Standard Costs file.
      * Process records that have an Item value.
      *
     C                   dou       *in90 = *on                                  Main do
     C                   read      tfp244                                 90
     C                   if        *in90 = *off and u1item <> 0                 If not EOF
      *
      * Retrieve the SYNON record using a key of:
      *   Company (hardcoded to 360)
      *   Item    (moved from a 5,0 field in the upload file to a 7,0 workfield)
      *   Date    (from the upload file)
      *
     C                   z-add     u1item        wkitem
     C     key01         chain     poaxcpl1                           92
     C                   if        *in92 = *off                                 If hit
      *
     C                   select
      * Meat
     C                   when      ldupcd = 'M'
     C                   if        u1cost <> 0
     C                   z-add     u1cost        axfrpr                         Meat Cost
     C                   endif
     C                   if        u1other <> 0
     C                   z-add     u1other       axfcpc                         Yield %
     C                   endif
      * Packaging
     C                   when      ldupcd = 'P'
     C                   if        u1cost <> 0
     C                   z-add     u1cost        axedpr                         Packaging
     C                   endif
      * Labor
     C                   when      ldupcd = 'L'
      *      Incremental
     C                   if        u1cost <> 0
     C                   z-add     u1cost        axeepr                         Labor - Inc
     C                   endif
      *      Labor Total
     C                   if        u1other <> 0
     C                   z-add     u1other       axfupr                         Labor - Tot
     C                   endif
      * Ingredients
     C                   when      ldupcd = 'I'
     C                   if        u1cost <> 0
     C                   z-add     u1cost        axefpr                         Ingredients
     C                   endif
10906 * Other
  |  C                   when      ldupcd = 'O'
  |  C                   if        u1cost <> 0
  |  C                   z-add     u1cost        axegpr                         Other
10906C                   endif
      *
     C                   endsl
      *
     C                   update    @axcphi
     C                   endif                                                  If hit
      *
     C                   endif                                                  If not EOF
     C                   enddo                                                  Main do
      *
      * EOF processing
      *
     C                   seton                                        lr
      /eject
