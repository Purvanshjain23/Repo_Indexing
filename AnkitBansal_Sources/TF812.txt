      * PROGRAM:     TF812 - Determine if Date is in Closed Revenue Week

      * CREATED:     06/28/05

      * Function:    This function is called by Synon function(s).
      *                 Synon sends us a date in CCYYMMDD format
      *                 We check the Revenue record in TFP099-Control file
      *                 We send back:
      *                         Y=Yes when the incoming date is in a closed week
      *                         N=No  when the incoming date is in an open week
      *
      ****************************************************************
      * MODIFICATIONS:
      ****************************************************************
      * DATE      PROGRAMMER
      *
      /eject
      ****************************************************************
      * FILE SPECIFICATIONS
      ****************************************************************
      *
     Ftfp099    if   e           k disk
      *    Function control
      *
      /eject
      ****************************************************************
      * DEFINITION SPECIFICATIONS
      ****************************************************************
      *
      *---------------------------------------------------------------
      * CONSTANTS
      *---------------------------------------------------------------
      *
     D yes             c                   'Y'
     D no              c                   'N'
      *
      *---------------------------------------------------------------
      * STANDALONE FIELDS
      *---------------------------------------------------------------
      *
      *
      * Parm fields
      *
     D xxflag          s              1
     D xxccyymmdd      s                   like(fnledt)
      *
      *
      ****************************************************************
      * DATA STRUCTURES
      ****************************************************************
      *
      *---------------------------------------------------------------
      * Standard program status data structure
      *---------------------------------------------------------------
      *    externally defined as UTPGFR (record format: PGMDSR)
     D pgmds         esds                  extname(utpgfr)

      *

     C                   if        sdpgm = *blank
     C                   endif
      /EJECT
      ****************************************************************
      * Mainline
      ****************************************************************

      *
     C     'REVENUE   '  chain(n)  tfp099                             92
     C                   select
      *
      * No record (this should never happen)
     C                   when      *in92 = *on
     C                   move      *blank        xxflag
      * Open
     C                   when      xxccyymmdd > fnledt
     C                   move      no            xxflag
      * Closed
     C                   when      xxccyymmdd <= fnledt
     C                   move      yes           xxflag
     C                   endsl
      *
     C                   seton                                        lr
      /eject
